FPGA Tank Game (TANKS)

Team Members: Dan Humeniuk, Melvin Bosnjak, Kyle Price, and Jaden Simon

Project Summary

TANKS is a top-down 2-player battle game ran on an Altera/Intel Cyclone V FPGA. Players use SNES controllers to fight to the death as an armored, rolling patriot of the USSR or the USA!

Our game was built from the ground-up, requiring the creation of a 16-bit CPU, a graphics pipeline, an input controllers, and a plethora of testbenches designed in Verilog. The game logic is written in our own assembly code, which is translated into machine code that can be ran by our CPU.

Key Features

  • Fully custom 16-bit CPU written in Verilog
  • VGA graphics stack supporting sprites, text and background layers
  • Start menu with multiple appeareance options for players and projectiles
  • Destructable walls
  • Collision and bounds detections
  • Sprite animations
  • SNES Controllers connected directly to the FPGA
