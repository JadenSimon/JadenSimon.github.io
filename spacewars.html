Space Wars

Team Members: Boston Clark and Jaden Simon

Project Summary

Space Wars is a C# recreation of the classic arcade game of the same name. Object-orientated programming techniques were used heavily throughout this project to allow for easier implementation of additional features beyond what the original game contained.

In addition to the game, a rudimentary website and database was created to view past games played as well as the highest scoring players (pictures currently unavailable).

Key Features

  • Multiplayer using UDP (tested up to 50 AI players)
  • Leaderboard showing player scores and health
  • Past games are saved and viewable through a website
  • Particle effects system
  • Global physics interaction (stars, ships, projectiles)
  • New game modes (free for all, teams, capture the star (CTS))
  • Customizable server settings through XML file

My Contributions

  • Designed the core game logic code
  • Created the particle effects and physics systems
