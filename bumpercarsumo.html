Bumper Car Sumo

Team Members: Dan Humeniuk, Melvin Bosnjak, Jaden Simon

Project Summary

Bumper Car Sumo is a 4-player robot battle game where the victor is the last one remaining in the arena. Players must stand on Wii balance boards to control their robots, giving them a unique and fun way to battle it out. To prevent players from cheating the system, a camera along with computer vision libraries detect the elimination of players and determine the winner.

Our robots are entirely our own design. This meant finding the necessary components that met our specifications as well as designing a printed circuit board (PCB) and 3D printed shell. An ESP-32 microcontroller was used as our core component given we need wireless functionality. Microcontroller code was written in C using the Arduino platform.

A Raspberry Pi is used as centralized server (known as "The Hub") that governs the robots and relays commands from the Wii balance boards to each robot. Server code was written in Python and OpenCV was used for computer vision functionality. All networking used UDP as the core protocol given its low latency.

Key Features

  • Custom robots and battle arena
  • Centralized control server
  • Robot detection using OpenCV
  • Supports 2-4 players
  • Wii balance board controllers
  • Smooth gameplay for up to 3 hours
